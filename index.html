Hello Educator!

This is a template for creating a codespace for your students. It includes a simple HTML page and a JavaScript file that you can use to teach your students how to use codespaces.

To get started, open the index.js file and start writing some JavaScript!

You can also edit the styles.css file to change the appearance of this page.

When you're ready to share this codespace with your students, click the "Share" button in the bottom left corner of the screen. This will copy a link to your codespace to your clipboard. You can then share this link with your students.

When your students open the link, they will be prompted to create a GitHub account if they don't already have one. Once they have a GitHub account, they will be able to access your codespace and start coding!
